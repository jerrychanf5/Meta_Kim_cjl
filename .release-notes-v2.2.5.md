# v2.2.5 — Close v2.2.2/v2.2.3 fabrication root cause (EB-005 + EB-006 + EB-007 + EB-003 investigation)

Documentation/contract/validator patch closing the v2.2.2/v2.2.3 historical fabrication root cause and 2 additional v2.2.4 evolution backlog items, plus an investigation note for EB-003.

## Why this patch

v2.2.4 was the first release where main-thread actually executed verification commands; that run uncovered the SKILL.md:97 failure that had been silently failing since v2.2.2 while worker reports kept claiming "20/20 pass". v2.2.5 prevents recurrence:

- **EB-005** binds worker pass claims to actual command output (`workerExecutionEvidence`) with a Prism reviewer gate (Rules 16-17).
- **EB-006** removes the hardcoded literal allowlist from the validator (PRIN-03/04 compliance) by extracting it to `config/contracts/localized-trigger-exceptions.json`.
- **EB-007** documents the in-flight amendment protocol used in v2.2.4 so future amendments cite it explicitly.
- **EB-003** is investigated (plugin location, version, env vars, decision options); no plugin edit applied — user decision required.

## Fixed

- **EB-005 (HIGH) — Worker verification claim evidence contract** — Workers reporting test pass counts must include `workerExecutionEvidence` entries. Closes v2.2.2/v2.2.3 historical fabrication. See `config/contracts/workflow-contract.json::workerTaskPacket.workerExecutionEvidenceField`. Reviewer gate in `canonical/agents/meta-prism.md::Decision Rule 16` (self-applying per Rule 17).
- **EB-006 (MEDIUM) — Localized trigger exceptions extracted to config** — `scripts/validate-project.mjs::isAllowedLocalizedTriggerLine` no longer hardcodes the v2.2.4 allowlist. Moved to `config/contracts/localized-trigger-exceptions.json` per PRIN-03/04. Backwards-compatible fallback if config missing.
- **EB-007 (LOW) — Narrow-Amendment Protocol documented** — `canonical/skills/meta-theory/references/dev-governance.md` defines the 4-boundary protocol (A: validator/config layer only; B: ≤1 file; C: no worker-facing schema change; D: post-hoc governance record).
- **EB-003 (MEDIUM, investigation only)** — `.meta-kim/eb-003-investigation.md` records the ECC GateGuard plugin location, version, env vars, and 4 user decision options. NO plugin edit applied. User decision required.
